语义化¶

<!Doctype html>

为关键链接添加 accesskey:

<a href="" title="" accesskey="M" rel="">Link</a>

abbr应该添加一个 title 属性对缩写进行描述:

<abbr title="Web Developer" >WD</abbr>

大段引用: <blockquote>，一般引用: <cite>

删除：<del>

定义列表：<dl>

格式化片段 <code>/<pre>，<pre> 的范围更广

强调: strong > em ≈ cite

表单项:<label for=”firstname”>

<img src=”” alt=”替代文本” />
