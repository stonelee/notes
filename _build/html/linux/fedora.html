技巧¶

fedora启动失败¶

启动fedora，提示:

Kernel panic - not syncing: VFS: Unable to mount root fs on unknown-block(0,0)

更换其他内核可以进入

查看/bin/grub2/grub.cfg, 发现最新内核下少了initrd/boot/initramfs-*.img

重新生成img:

$ yum reinstall kernel

Fedora 10里将普通用户添加到sudo组¶

  1. 在终端输入su -
  2. 输入密码， 这样就切换到root了
  3. 输入visudo
  4. 找到 root ALL=(ALL) ALL 在这一行下边按a键进入编辑模式，然后输入： yourname ALL=(ALL) ALL,然后按esc退出
  5. 按 :wq 键保存退出
  6. 输入exit退出root权限
  7. 测试，在当前用户下，输入sudo whoami

如果一切正常，命令会返回 “root” 这个字。

fedora启动时显示启动信息¶

  1. 编辑/etc/default/grub， 去掉GRUB_CMDLINE_LINUX中quiet和rhgb
  2. /sbin/grub2-mkconfig -o /boot/grub2/grub.cfg
  3. 查看/boot/grub2/grub.cfg，启动配置中已经没有了quiet，rhgb选项。
